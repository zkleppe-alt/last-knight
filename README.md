# Kingdom's Last Knight

A single-file HTML5 canvas RPG. Boot through the GBA-style intro, pick a save slot, and play.

## ▶ Play

**https://YOUR-USERNAME.github.io/last-knight/**

*(This link goes live once GitHub Pages is enabled — see below. Replace `YOUR-USERNAME` with your GitHub username.)*

## Controls

- **Arrow keys / WASD** — move
- **A button / Z / J / Enter** — confirm, talk, attack
- **B button / X / K / Esc** — cancel, back
- **☰ / M** — open the menu
- On phones, use the on-screen touch controls.

## Saving

The game saves to your browser (three slots). Because saves live in the browser, they stick as long as you play from the same link on the same device. To move a hero between devices, use **☰ → System → 📤 Export Save Code** and **📥 Import** on the other device.

## Feedback

Playing and hitting a bug? Open an **Issue** on this repo (the *Issues* tab up top) with what you were doing when it happened. Screenshots help.
